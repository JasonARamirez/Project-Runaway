POST: roadTrip

Purpose Allows user to create road trip with a start location, end locations, and start time; returns Google Maps directions with safest route for the road trip.
URL The URL structure
https://project-runaway.herokuapp.com/roadTrip
Method POST
URL Params Required:
request=[POST roadTrip json]
Data Params POST roadTrip json
Content: 
request : {
  userID : [given API Key],
  start :[[latitude], [longitude]],
  end : [[latitude], [longitude]],
  startTime : [day]/[month]/[year]-[hour]:[minutes]:[seconds]
}
Example: 
request : {
  userID : "45a759cf-4305-U5bf-a0a2-a1196f978d76",
  start :[42.032194, -93.638217],
  end : [38.638800, -90.231728],
  startTime : "24/03/2017-12:00:00"
}
Success Response Code: 200
Content: 
{
  success : 1,
  routeID : [String],
  route : {
    paths : [
      {
        distance : [Float in Miles],
        polyline : [String for Google Maps],
        start : [[Latitude],[Longitude]],
        end : [[Latitude],[Longitude]],
        numPath : [Integer],
        time : [Integer in Seconds]
      }
      ...
    ],
    start : [[Latitude],[Longitude]],
    end : [[Latitude],[Longitude]]
  }
}

Content Example: 
{
  success : 1,
  routeID : "55e781fc-0994-R0c9-8bf1-02412b4e067a",
  route : {
    paths : [
      {
        distance : 0.164727799,
        polyline : "elp_GzuozPa@EiBSeBQwCYgAMMC",
        start : [42.032194,-93.6382174],
        end : [42.0345942,-93.6377758],
        numPath : 1,
        time : 28
      }
      ...
    ],
    start : [42.032194,-93.638217],
    end : [38.6388,-90.231728]
  }
}
Error Response Content: { success : 0, error : [String Message]}
